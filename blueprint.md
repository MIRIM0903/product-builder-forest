
# Blueprint

## Overview

This document outlines the plan for creating a website with the theme "고민을 행동으로 바꿔드립니다" (We turn your worries into action).

## Current Plan

### Phase 1: Initial Setup

1.  **Update `index.html`**: Create the basic HTML structure for the new website.
    *   Set the title to "고민을 행동으로 바꿔드립니다".
    *   Add a header, main, and footer section.
    *   Include a hero section with a headline and a call-to-action button.
2.  **Update `style.css`**: Style the new HTML structure.
    *   Apply a modern and clean design.
    *   Use a professional color palette and fonts.
    *   Ensure the layout is responsive.
3.  **Update `main.js`**: Add interactivity to the site.
    *   Implement smooth scrolling for anchor links.
    *   Add a particle effect to the hero section for visual appeal.

## New Plan: Lotto Number Generator

1.  **Update `index.html`**: Change the structure to a lottery theme.
    *   Set the title to "Lotto Number Generator".
    *   Add a main heading.
    *   Add a button to generate numbers.
    *   Add a container to display the numbers.
2.  **Update `style.css`**: Adjust the styling to match the new lottery theme.
    *   Center the content.
    *   Style the generate button.
    *   Style the number display area with colored circles for each number.
3.  **Update `main.js`**: Implement the lottery number generation logic.
    *   Generate 6 unique random numbers between 1 and 45.
    *   Sort the numbers.
    *   Display the numbers in the styled container.
